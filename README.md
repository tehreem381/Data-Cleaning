This repository contains my work and practice exercises from the Kaggle Learn: Data Cleaning course.
The course focuses on handling messy, incomplete, and inconsistent datasets — a critical skill in real-world machine learning and data analysis projects.

Objectives
During this course, I learned how to:
Handle missing values
Scale and normalize data
Parse and standardize dates
Fix inconsistent text data
Work with character encodings

Topics Covered

1️⃣ Handling Missing Values
Detecting missing data
Removing rows/columns
Imputation techniques
Understanding when to drop vs fill

2️⃣ Scaling & Normalization
Min-Max scaling
Standardization
Log transformation
Handling skewed distributions

3️⃣ Parsing Dates
Converting text to datetime
Extracting day, month, year
Handling inconsistent date formats

4️⃣ Character Encodings
UTF-8 encoding issues
Detecting encoding errors
Fixing corrupted text

5️⃣ Inconsistent Data Entry
Standardizing text fields
Fixing typos
Cleaning categorical data
